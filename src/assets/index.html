s/o/cute

- the universal discord bot -

The discord 'bot' that's not a bot

s/o/cute takes advantage of the discord substitute command feature to allow you to use commands without the need for a bot. This means that you can use the commands in any server, without the need for a bot to be added to the server.

Advanced Image Manipulation

s/o/cute has a wide range of image manipulation commands, such as adding text, changing the color of an image, and more.

Fun Commands

s/o/cute has a lot of fun commands, such as a meme command, commands to get random animal pictures, and more.
